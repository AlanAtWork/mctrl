HTML Document

This basic HTML document lives as a resource embedded in the program. The HTML control is able to display such documents. It can be pointed with the URL res://ex_html.exe/1000

The document can use other resources as images or cascading style sheets as the image below demonstrates.

Of course it can also handle normal HTTP, HTTPS and other protocols:

  • http://mctrl.sf.net/ - Of course the control supports HTTP and HTTPS protocols.

Finally you can have links which are handled programatically by the application code. When you click on such link, the parent window of HTML control gets notification.
